## Project

**Cora AI OS** — a governed multi-agent assistant + AI operating system on self-hosted infra
(Ubuntu Server, Docker, Postgres/pgvector, Redis, n8n, DGX Spark). Cora is the user persona;
ATLAS is the internal orchestrator; specialist subagents are SCRIBE (memory), FORGE (engineering),
PULSE (research/web search), SIGNAL (communication), CHRONOS (time/schedule). Non-autonomous by
design: deterministic routers + manual triggers + admin actions only; everything auditable.

Repo `/home/owner/cora-ai-os/` (cora-api, cora-ui, cora-worker, mcp-filesystem, mcp-placeholder);
infra compose `/home/owner/cora-stack/` (n8n + redis). Stack deployed and healthy as of 2026-06-12.

Audited (`calendar_access_events`) + traced. Per-provider OAuth redirect derivation lets gmail + google_calendar coexist.
